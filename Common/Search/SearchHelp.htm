Common Search

Many Search iPart settings can be overwritten in a URL parameter by using the Content Item Key (GUID). The URL parameter override affects all Search iParts on a page.

  • Name - Enter a name for the content item. This text appears during design mode (configuration).
  • Title - Enter the text to display above this iPart on the rendered page. If blank, no title displays.
  • Do not render in design mode - Select this option to prevent content record data from being loaded in the content editor. This improves performance during design time (configuration).
  • Display a border around this content - Select this option to display a border around this iPart on the rendered page.

Search Area Settings - hides or shows the Search field, to let users broaden or narrow results

  • Search Label - Enter the text to display next to the search field. This can be overwritten by a URL parameter (attribute name = SearchLabel).
  • Display Search Area Options - Select the items to display in the Search Area drop-down list. This can be overwritten by a URL parameter (attribute name = DisplayAreaOptions plus GUID for area as displayed in Design view).
  • Options to Display - Enter a comma-separated list of the Search Area options to display in a drop-down list for user selection. The All selection displays all content records. If no options are selected, the drop-down list does not display and the Initial Search Area is used for display. This can be overwrittenby a URL parameter(attribute name = Options plus GUID for area as displayed in Design view).
  • Display Search Text Box - Select this option to display a Search text box. This can be overwritten by a URL parameter (attribute name = DisplayTextBox).

Initial/Default Display Settings - skips or sets initial settings, to narrow results to an area or value

  • Run Search for Initial Display - Select this option to automatically run a specified search on page load for the user's initial view. If not selected, search is not run until the user clicks the Go button. This can be overwritten by a URL parameter (attribute name =InitialDisplay).
  • Initial Search Area - Select a search area to display when Run Search for Initial Display is selected or when Display Search Text Box is not selected. Only one area may be selected. This can be overwritten by a URL parameter (attribute name = InitialArea).
  • Initial Search Text - Enter the text filter for search results when Run Search for Initial Display is selected. This can be overwritten by a URL parameter (attribute name = InitialText).
  • Initial Search Text Logic - Specify the logic to be applied to multiple text filters and selected tags when Run Search for Initial Display is selected. For example, you can specify "Committee And Chair" or "Committee Or Chair" to return the appropriate search results. The default value is And. This can be overwritten by a URL parameter (attribute name = InitialLogic).

Add New Item Settings - shows an Add link to create related content

  • Content Item for Add - Enter the content item code to display an Add link for the content item. This property is enabled only if: Initial Search Area is not "All" And Display Search Area Options is disabled. This can be overwritten by a URL parameter (attribute name = AddContentItem).
    Note: Allowed URL values are Content Item Key (GUID) or Content Item Code (string).

Search Results Settings - controls how search results and messages appear to users

  • No Results Found Message - Enter text for the message to display if no results are found. The default value is set by the system parameter SE.NoResultsMessage. This can be overwritten by a URL parameter (attribute name = NoResultMsg).
  • General Error Message - Enter text for the message to display if search failed. The default value is set by the system parameter SE. Include '{0}' to display the specific system error message. This can be overwritten by a URL parameter (attribute name = ErrorMsg).
  • Results to Display per Page - Enter the number of results to display per page. 0 = display all results. The default value is set by the system parameter SE. This can be overwritten by a URL parameter (attribute name = PageSize).
  • Maximum Results to Display - Enter the maximum number of results allowed for display. The default value is set by the system parameter SE. This can be overwritten by a URL parameter (attribute name =MaxResults).
  • Maximum Results Exceeded Message - Enter the message to display if the number of results exceeds the maximum allowed by Maximum Results to Display. The default value is set by the system parameter SE.ExceedMax. This can be overwritten by a URL parameter (attribute name = ExceedResultsMsg).
  • Sort Order - Select the sort order method for search results. The date options are based on the Publication Date/Created Date/Start Date in the item's metadata. This can be overwritten by a URL parameter (attribute name = Sort).
  • Selected Tags - (optional) Select tags to apply to content created in this iPart. Tags are helpful for a user to filter search results and for an administrator to identify content. This can be overwritten by a URL parameter (attribute name = Tags plus GUID for area as displayed in Design view).
